大明寺筹建戒坛缘起

大明寺始建于南朝宋孝武帝大明年间，故称大明寺。天宝元年间高僧鉴真于寺中讲律传戒。天宝元年（公元753年）东渡日本，传戒弘法，成为日本开山祖师。

律宗或律学是以佛教经典《四分律》为准则并会通大乘教义而成的独立宗派，其创始人道宣（596-667），在终南山寺弘《四分律》并创立戒坛，制定佛教的授戒仪制，被称为“南山律宗”。鉴真正是授传于这一派大师道岸，弘景，并东传日本而开创日本律宗。由于历史的种种原因，战争、灾害、大明寺戒坛早已不复存在。鉴真第八十三世传人能修大和尚发心修建大明寺戒坛。

建戒坛是对鉴真律学文化的传承，是对大明寺律宗第一大寺地位的体现，是对当今时代佛教能够长期健康发展下去有着重要的意义。
